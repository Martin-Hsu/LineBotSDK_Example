## 使用LineNotify  
有以下函數可以使用:

```
GetToeknFromCode -> 使用code來取得token  
  
SendNotification (message image sticker) -> 發送notify給使用者  
  
GetStatus -> 取得token狀態  
  
RevokeToken -> 註銷token來取消連動  
  
GenerateHTMLString -> 產生可使用的html程式碼  
```
